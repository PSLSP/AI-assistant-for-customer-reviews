# Примеры аналитики и структура отчёта по отзывам

> **Данные вымышленные** (учебный проект, магазин «ТестоМаркет»).

---

## 1. Краткая сводка за период (пример текста для руководителя)

**Период:** 1–7 марта 2026 (тестовые данные)

За неделю обработано **42** отзыва. Распределение по тональности: позитив **55%**, нейтраль **24%**, негатив **21%**. Доминирующие темы: доставка (**38%** всех тем), качество товара (**27%**), работа консультантов (**18%**), остальное — цена, возвраты, сайт.

**Повторяющиеся проблемы (топ-3, вымышленно):**  
1. Задержки курьера на 1–2 дня в регионе «Север-тест».  
2. Претензии к упаковке хрупких товаров.  
3. Долгий ответ в чате в вечерние часы.

**Рекомендации (логика, не приказ):** усилить коммуникацию по задержкам; чеклист упаковки для категории «стекло/электроника»; сдвиг смены или автоответ в чате после 20:00.

## 3. Структура еженедельного отчёта (шаблон разделов)

1. **Объём и источники** — сколько отзывов, откуда (сайт, маркетплейс, соцсети — если есть).  
2. **Тональность** — доли или индекс удовлетворённости (как принято в компании).  
3. **Тематика** — топ тем, динамика к прошлой неделе.  
4. **Повторяющиеся проблемы** — формулировки «что говорят клиенты» + гипотеза причины.  
5. **Позитивные инсайты** — что хвалят чаще всего (для маркетинга и обучения).  
6. **Действия** — что сделано / запланировано (поддержка, логистика, продукт).  
7. **Риски и эскалации** — юридические, безопасность, СМИ (если были).

```
Ниже пять ВЫМЫШЛЕННЫХ отзывов о магазине «ТестоМаркет».

(3) два предложения для улучшения сервиса. Не используй реальные имена и адреса.

[вставить отзывы]
```

## 5. Мини-пример «сырых» отзывов для учебной сводки

1. *«Заказ ТМ-1001, курьер опоздал на два дня, никто не предупредил.»*  
2. *«Ваза пришла с трещиной, упаковка тонкая.»*  
3. *«Консультант Анна (вымышленное имя) помогла с размером — спасибо!»*  
4. *«Дорого по сравнению с конкурентом, но доставка быстрая.»*  
5. *«Опять задержка доставки, уже второй раз за месяц.»*

Ожидаемая логика аналитики: пересечение тем «доставка» + негатив; «качество/упаковка»; позитив по персоналу.
