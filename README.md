# ИИ-ассистент для работы с отзывами клиентов

Данный проект реализует нейроассистента, который поможет быстро и грамотно обрабатывать отзывы клиентов, формировать корректные ответы и выявлять повторяющиеся проблемы.
Проект включает в себя создание системного промпта, базы знаний в Google Sheets и документации для настройки ассистента в **ChatGPT** или **OpenAI Playground**. Код приложения не требуется — только текстовые артефакты.

## Цели и задачи

Ассистент помогает:

- классифицировать отзывы по **тональности** и **теме**;
- генерировать **корректные ответы** на отзывы пользователей с использованием базы знаний;
- готовить **сводки и аналитику** по темам и оценкам;
- снизить нагрузку на подразделения поддержки и маркетинга.

Сценарии: **жалоба**, **благодарность**, **предложение**.  
Ограничение: ассистент **не публикует и не удаляет** сообщения на площадках, только готовит текст для оператора.

## Как протестировать ассистента

### 1. Предварительная подготовка

#### 1.1. Импортировать базу знаний и примеры запросов клиентов из Google Sheets в PDF-файлы.

#### 1.2. В [OpenAI Playground](https://platform.openai.com/chat) добавить в векторное хранилище импортированные PDF-файлы:

- Выбрать **Tools → Add → File Search**
- В открывшемся окне загрузить PDF-файлы и задать имя векторного хранилища
- Дополнительные параметры хранилища оставить по умолчанию
- Нажать **Attach** — создастся векторное хранилище с заданным именем

---

### 2. Тестирование

#### 2.1. Ответ ассистента на запрос пользователя по шаблону из базы знаний

**Пользовательский промпт:**

```text
Вот отзыв:
"Заказ ТМ-1001, курьер опоздал на два дня, никто не предупредил."
Классифицируй и предложи ответ. Если шаблон ответа найден в базе знаний, то используй его.
```

**Ответ ассистента** (формулировка ответа берётся из шаблона):

- **Классификация:** негатив + доставка.

- **Формулировка ответа клиенту:**

> Спасибо за ваш отзыв. Нам очень жаль, что доставка прошла не так, как вы ожидали. Мы передадим информацию в службу логистики для проверки. Пожалуйста, сохраните данные заказа — по ним мы сможем уточнить статус.

**Кратко:** требуется улучшение коммуникации по задержкам доставки. Отправить в службу логистики.

*Источник в тесте:* База знаний для теста.pdf

---

#### 2.2. Ответ ассистента на запрос пользователя без использования шаблона

**Пользовательский промпт:**

```text
Вот отзыв:
"Решил воспользоваться вашим сервисом. Пока непонятно, что к чему, как разберусь - напишу".
Классифицируй и предложи ответ. Если шаблон ответа найден в базе знаний, то используй его.
```

**Ответ ассистента** (шаблон ответа не найден; формулировка в соответствии с форматом системного промпта):

- **Классификация:** нейтраль (нейтрал) + сервис

- **Формулировка ответа клиенту:**

> Спасибо за ваш отзыв! Мы рады, что вы решили воспользоваться нашим сервисом. Если у вас возникли вопросы или нужна помощь в процессе, пожалуйста, не стесняйтесь обращаться к нам. Мы готовы ответить на любые ваши запросы.

**Кратко:** клиент испытывает неопределённость с сервисом. Следует быть готовыми предоставить помощь и ответить на возможные вопросы.

##### Отчёт NPS

**Общая статистика по NPS**

| Тип клиента | Количество отзывов | % |
|-------------|--------------------|---|
| Промоутеры  | 4                  | 33% |
| Нейтралы    | 2                  | 17% |
| Детракторы  | 6                  | 50% |
| **Итого**   | **12**             | **100%** |

**NPS**

`NPS = % Промоутеров − % Детракторов = 33% − 50% = −17`

##### Рекомендации
