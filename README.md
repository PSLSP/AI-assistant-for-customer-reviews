# ИИ-ассистент для работы с отзывами клиентов

Данный проект реализует прототип нейроассистента, который поможет быстро и грамотно обрабатывать отзывы клиентов, формировать корректные ответы и выявлять повторяющиеся проблемы.
Проект включает в себя создание системного промпта, базы знаний в Google Sheets и документации для настройки ассистента в **ChatGPT** или **OpenAI Playground**. Код приложения не требуется — только текстовые артефакты.

**English:** Educational prompt-engineering project: a system prompt, a CSV knowledge base, and docs for ChatGPT / OpenAI Playground. No application code — artifacts only.

## Цели и задачи

Ассистент помогает:

- классифицировать отзывы по **тональности** и **теме**;
- генерировать **корректные черновики ответов** (публикацию выполняет человек);
- готовить **сводки и зачатки аналитики** по темам и оценкам;
- снизить рутину для поддержки и маркетинга (в учебной постановке).

Сценарии: **жалоба**, **благодарность**, **предложение**. Ограничение: ассистент **не публикует и не удаляет** сообщения на площадках — только готовит текст для оператора.

## Требования

- Аккаунт **OpenAI** (Playground) и/или **ChatGPT** для проверки сценариев.
- Редактор текста или Google Docs / Notion — по желанию для оформления сдачи курса.
- Для таблицы: **Google Таблицы** или любой редактор CSV (кодировка **UTF-8**).

## Быстрый старт

1. Откройте [docs/Системный_промпт_ассистент_отзывы.md](docs/Системный_промпт_ассистент_отзывы.md) и скопируйте блок **«Текст для вставки в ChatGPT / Playground»** в **Custom Instructions** (ChatGPT) или поле **System** (Playground).
2. Базу знаний в Playground **нельзя** подключить по URL: скопируйте содержимое [data/knowledge_base.csv](data/knowledge_base.csv) в **первое сообщение User** (подробно в [Инструкция_по_обновлению_ассистента.md](docs/Инструкция_по_обновлению_ассистента.md), раздел *OpenAI Playground*).
3. Далее в User присылайте вымышленные отзывы и формулируйте задачи (классификация, ответ, сводка).

## Экспорт в Google Docs / Таблицы

- **Docs:** вставка из Markdown вручную или импорт через конвертацию в `.docx`, затем «Файл → Открыть» в Google Диске.
- **Таблицы:** «Файл → Импорт → Загрузка» → выберите `knowledge_base.csv`, разделитель **запятая**, кодировка **UTF-8**.

## Персональные данные

В материалах используются **только тестовые и вымышленные** данные (в т.ч. бренд «ТестоМаркет»). Реальные отзывы, ФИО, телефоны и прочие ПДн в репозиторий **не добавляйте**.
